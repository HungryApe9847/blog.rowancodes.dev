# Top 5 Minecraft Sites!
## 5:
### Chunkbase (or any other seed visualiser) (https://chunkbase.com/apps)
Pros:
- Good for finding stuff like slime chunks.

Cons:
- Can seem complex at first.
- Can be considered cheating by some.
## 4:
### NameMC Skins (https://namemc.com)
Pros:
- The tags feature is quite useful, helped me find some ideas and bases for my skin.

Cons:
- Search doesn't do specific skins, it searches for skin tags.
- Has less skins than skindex.
## 3:
### Skindex Skins (https://www.minecraftskins.com/)
Pros:
- Skindex is a MASSIVE place for Minecraft skins

Cons:
- A lot of skins are not in a style I would like, as there are so many.

## 2:
### Curseforge (https://www.curseforge.com/)
Pros:
- Massive modding site.
- Has the best mods like <a href = "https://www.curseforge.com/minecraft/mc-mods/jei">JEI</a>, <a href = "https://www.curseforge.com/minecraft/mc-mods/jade">Jade</a> and more!
- Modpack feature is great for some who want a preset experience.

Cons:
- Doesn't have Optifine.
- The desktop installer is better than the manual site.
- Some people don't use mods. (Obviously not me.)
## 1:
### DigMinecraft Command Generator
Pros:
- Easy to use command generator.
- Great for learning commands as a beginnner.

Cons:
- Hard to continue learning after a point.
